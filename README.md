# WiFi Device Manager

WiFi Device Manager is a Flutter desktop application designed to detect, analyze, and manage devices connected to a WiFi network. This application provides users with an intuitive interface to view and manage their network devices effectively.

## Features

- **Device Detection**: Automatically detects devices connected to the WiFi network.
- **Device Analysis**: Provides detailed information about each connected device, including IP address, MAC address, and device type.
- **Device Management**: Allows users to manage connected devices, including options to block or prioritize devices.

## Getting Started

To get started with the WiFi Device Manager, follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/yourusername/wifi_device_manager.git
   ```

2. Navigate to the project directory:
   ```
   cd wifi_device_manager
   ```

3. Install the dependencies:
   ```
   flutter pub get
   ```

4. Run the application:
   ```
   flutter run
   ```
